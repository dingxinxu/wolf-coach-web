# 狼人杀高手实战策略手册

> 基准板子：12 人预女猎守 / 预女猎白，屠边制。

## 核心心法（永远先看这十条）

1. **预言家是逻辑起点**：警徽流是灵魂，发言饱满度决定信任度。
2. **女巫是胜负手**：解药抢轮次，毒药定胜负，**盲毒是大忌**。
3. **猎人威慑大于开枪**：携枪归票，被毒不能开枪是硬伤（见 `rules.md` §四）。
4. **守卫是隐形核心**：空守为主，配合女巫打双神盾，不要过早暴露。
5. **白痴是被动神**：翻牌后归票权最大，残局扛推为好人争轮次。
6. **狼人配合大于个人**：四大角色（悍跳/冲锋/倒钩/深水）分工明确，统一口径。
7. **行为 > 票型 > 逻辑 > 发言**：发言会骗人，票型可能是假的，**行为不会**。
8. **变票最值得关注**：变票背后的心路历程是找狼的关键。
9. **不要弃票**：除非有明确战术目的，跟票坐定好人更安全。
10. **预言家第一夜被刀 + 女巫没救 = 好人原地爆炸**——所以女巫首夜救人是标准打法。

## §1 预言家（最关键的神职）

预言家是好人逻辑起点，**没有自证能力、没有自保能力**，是"全场最弱的神"，也是被狼悍跳对冲的第一目标。预言家水平直接决定好人胜率。

### §1.1 第一夜验人后的发言模板

**（1）标准节奏式（最稳）**
结构：起跳身份 → 报查验 → 留警徽流 → 聊心路历程 → 点评前置位 → 总结归票。

> "我是真预言家，昨晚验的 X 号，是金水/查杀。"
> "警徽流先留 A，再留 B。"（说明理由）
> "我为什么验 X：因为我警上想验警下大概率上警的牌 / 想验发言位的牌。"
> "前置位的 Y 号发言我听着像狼/像好人，理由是……"
> "希望大家把警徽投给我，我打出警徽流带好人找出四狼。"

**（2）抿人式发言（高配，后置位发言适用）**
先不立刻报全部信息，边听前置位发言边在心里重新定义，最后再用饱满的心路历程打动好人：
- 先点评前置位两张可疑牌的发言漏洞
- 再报出自己的查验和警徽流
- 重点解释"我从 X 的警上发言中嗅到了狼味，所以警徽流第一张打 X"
- 抿人式更易让好人相信你有"真预言家视角"

**（3）谨慎式（验出查杀、位置不利、预判后置位会悍跳）**
- 先低调报验人
- 不急于拉仇恨，留警徽流避开后置位可能的悍跳狼
- 留余地："如果后置位有人对跳，我警徽流会做相应调整。"
- 把火力集中给查杀牌

### §1.2 警徽流怎么留（核心技术）

**基本逻辑**：
- "验 A 留 B"：今晚验 A，若 A 金水（好人），死后警徽给 B；若 A 查杀（狼），警徽撕掉或外置位飞给公认好人。
- "验 A 弃 B"：A、B 同时是警徽流，验 A 金水则飞 A，验 A 查杀则飞 B。

**标准留法**：
> "今晚验 A，明天验 B。A 金水警徽飞 A，A 查杀警徽飞 B（再验 B）；A、B 双金水撕警徽（外置位大概率有狼）；A、B 双查杀警徽外飞。"

**留谁的原则**：
1. **优先留狼坑**：把警徽流砸在你认为最可能是狼的位置。
2. **警上多狼 vs 警下多狼判断**：警上发言多漏洞偏向警上；悍跳狼给警下发查杀，警徽流必留警下（被查杀那张一定会上警给你投票，是关键票）。
3. **不要留队友（金水好人）做警徽流**：浪费验人，且容易被狼队自爆吞信息。
4. **留能上票给你的牌**：让警下关键牌感受到压力，拉票同时软定义其身份。

**防自爆警徽流（高配）**：在有守卫的板子中可流三天，防止狼队连续自爆吞掉所有验人信息（双爆吞警徽机制见 `rules.md` §三.3）。

### §1.3 被狼悍跳对冲时如何识别演技

| 硬指标 | 真预言家 | 悍跳狼 |
|---|---|---|
| **视角** | 闭眼视角（只知验人） | 睁眼视角（知队友是谁） |
| **点评准确度** | 有不确定感 | 过度准确/过度攻击 |
| **警徽流走向** | 指向狼坑 | 暴露拉票心理（全留给站真预的好人） |
| **力度** | 坚定、有细节 | 模仿模板但缺个性化分析 |
| **位置感** | 对多个对跳者敌意有差别 | 一视同仁攻击往往不正常 |

**金水反应**：被发金水的牌如果"反水立警"（不接金水自己跳预言家），往往意味着狼队玩花套路。

#### 倒钩狼识别清单（实战行为信号）

悍跳狼好识别，**倒钩狼才是预言家最难抓的狼**——他铁站你边、攻击你的对手、心路听起来饱满。识别倒钩狼靠**行为细节**，不是逻辑：

| 行为信号 | 倒钩狼面 | 解释 |
|---|---|---|
| **事后诸葛亮**（"我早看 X 像狼"，但之前发言没提过） | ⭐⭐⭐⭐⭐ | 真好人会在第一时间表态，事后补刀的往往是补身份的狼 |
| **主动归票拉仇恨**（深水狼本应跟票不该归票） | ⭐⭐⭐⭐ | 铁站边的好人突然主动归票 = 想扛推别人 = 视角暴露 |
| **攻击队友"逻辑"过于精准** | ⭐⭐⭐⭐ | 倒钩狼踩队友会踩在"恰好不伤及团队核心"的点上，太精准=睁眼视角 |
| **站边时机过早且无悬念** | ⭐⭐⭐ | 真好人会犹豫、会等更多发言；铁站不犹豫的多半是队友配合 |
| **从不当焦点、从不被归票** | ⭐⭐⭐ | 全程划水的"铁好人"牌，决赛圈往往是最后一只狼 |
| **票型始终和好人多数站一起** | ⭐⭐⭐ | 倒钩狼会上票真预、票悍跳狼——这是做高身份的核心手段，但票型看不出 |
| **抿女巫/抿猎人过度积极** | ⭐⭐⭐⭐ | 倒钩狼需要为狼队找神，会过度试探"谁是神" |

**抓倒钩狼的口诀**：**"好人会犹豫，狼人会坚定；好人归票慎重，狼人归票积极；好人细节模糊，狼人细节精准。"**

**教练给用户的提示模板**：
> "X 号虽然站你边，但有几个细节要警惕：① 他今天才说'早看 8 号像狼'（昨天没提，事后诸葛亮）；② 他主动归票 9 号（深水狼不该归票）。**建议你（预言家）今晚验他，不要轻信铁站边的好人**——9 人局倒钩狼是主流狼战术。"

#### 话术精悍跳狼识别（心路过饱满反而是狼）

新手预言家容易凭"心路饱满度"判真伪，但**高配悍跳狼的心路比真预还饱满**——因为他们是睁眼视角，能编出更精细的位置博弈分析。识别要点：

| 信号 | 话术精悍跳狼 | 新手真预言家 |
|---|---|---|
| **心路细节** | 过于精细（位置博弈、概率分析、多重理由） | 粗略（"感觉他像狼"） |
| **点评准确度** | 每个位置都有准确定位 | 只点评相邻/明显位 |
| **警徽流理由** | 给出复杂理由（"X 位置如果被狼用来冲锋最舒服"） | 随机留（"留 8、9 吧"） |
| **不确定感** | 几乎没有（什么都讲得通） | 有犹豫（"我验 7 号是因为……嗯……"） |
| **模板化痕迹** | 像背过稿（起跳→报验→警徽流→心路→归票一气呵成） | 断断续续、有卡壳 |

**核心反直觉**：**"心路越饱满，越可能是准备过的狼"**。真预言家是闭眼视角，必然有犹豫和不确定；悍跳狼是睁眼视角，反而能讲得天衣无缝。

**教练识别口诀**：**"过度饱满=准备过；天衣无缝=睁眼视角；犹豫卡壳=闭眼真预。"**

**应对策略**（教练给好人/警长的提示）：
> "X 号心路太饱满了，饱满得有点刻意。真预言家是闭眼的，不可能每个位置都分析得这么精细——他连后置位没发言的牌都能讲出'位置博弈'，这是睁眼视角才有的信息量。**建议警下别急着投他**，听听对跳的 Y 号，Y 号虽然心路粗，但'粗'反而像真预。"

#### 中期倒钩狼预警（第二天起必查）

倒钩狼最大的伪装是"站对边"——票过悍跳狼队友、攻击过狼队友的逻辑。第二天起，**对"铁站真预边+票过狼队友"的牌必须警惕**，按下面清单排查：

**高危画像**（中后期倒钩狼的典型特征）：
1. 第一天铁站真预边（无悬念、无犹豫）
2. 票型票了悍跳狼（狼踩狼做高身份）
3. 攻击悍跳狼的逻辑"恰好不伤及狼队核心"
4. 从不当焦点、从不被归票
5. 全程"独立思考的中立好人"人设
6. 第二天起开始**软踩平民**（为决赛圈扛推铺路）

**教练中期预警模板**（给预言家/警长/女巫）：
> "现在场上 X 号的身份好得离谱——票过悍跳狼、铁站你边、从不当焦点。**这种'完美好人'正是倒钩狼的核心伪装**。建议你（预言家）下一警徽流砸他，或（女巫）把他列入毒药候选。不要因为他'帮过你'就排除他——9 人局倒钩狼帮真预就是为了做高身份。"

**关键提醒**：**"站对边 ≠ 好人**。倒钩狼的核心战术就是站对边。识破倒钩狼要靠**行为细节**（事后诸葛亮、主动归票、攻击过于精准），不是靠"他站我边"。

### §1.4 报查验结果的标准话术

- 金水："X 号是我昨晚验的人，**金水**（好人），请 X 号接水。"
- 查杀："X 号是我昨晚验的人，**查杀**（狼人），希望好人合力把 X 号投出去。"
- 报验人心路："我之所以验 X，是因为前置位/警下/某玩家发言让我感觉 X 可能是狼……"
- 警徽流交接说明："我若今晚倒下，验 A 金水警徽飞 A，验 A 查杀警徽撕/外飞。"

### §1.5 被女巫盲毒 / 被狼认下时怎么办

**被女巫盲毒（最惨）**：
- 第一时间明确报出全部验人信息和警徽流后续安排
- 若已是警长（首日白天已当选）：强烈呼吁**警徽移交给指定好人**（飞徽传递验人暗号，见 `rules.md` §三.4）
- 若尚未当选警长（首夜即被盲毒，新时序下死者不参与警长竞选）：呼吁**好人把警徽投给指定好人**，并请后续当选的警长在危急时按你的警徽流安排传递信息
- 提醒女巫："盲毒到真预言家是好人崩盘的开始。"

**被狼认下（狼故意站边你、做高你身份）**：
- 立刻识别"被狼认下"反而会脏自己身份（这是阴阳倒钩的核心手法）
- 公开声明："X 号站边我我并不感谢，请 X 号从我的团队里走开。"
- 主动把狼"塞进来"的牌踢出去，避免被打成狼团队。

### §1.6 藏预言家（虚晃，慎用）

- **警上诈身份**：极少数高玩会警上不立刻跳，等听到悍跳狼发言漏洞后再起跳。风险极大（女巫解药可能被骗、警徽可能丢失），**不推荐萌新使用**。
- **末位发言保留信息**：后置位发言时，先把分析做足，最后一刻再报验人，制造信息密度。
- **不暴露验人逻辑给狼队**：报验人心路时不要过于详细暴露你下一步要验谁。

## §2 女巫（胜负手）

女巫水平直接代表玩家整体水平。一瓶解药抢轮次，一瓶毒药定胜负。

> 规则前提：女巫双药机制、首夜看刀口规则、不能自救——详见 `rules.md` §二.2 + §五.2。

### §2.1 第一夜是否救人

**标准结论：首夜应该救**。理由：
1. 狼人首刀刀中神的概率约 1/2，刀中预言家最致命。
2. 即使狼人自刀骗药，救起后好人仍**抢回一个轮次**，不会吃亏。
3. 救人后手里还有毒药，可以观察银水（被救起的人）后续表现，发现问题用毒药补救。

**不救的情况**：
- 熟人局/高端局，为防暴露规律故意不救，把解药留给预言家。
- 自己被首刀：女巫不能自救（`rules.md` §五.2），只能选择是否盲毒。
- 守卫板子中：守卫空守期望女巫盲救形成首夜平安夜，避免同守同救（奶穿，见 `rules.md` §五.2）。

### §2.2 解药和毒药的使用时机

**解药**：
- 首夜用：标准打法，抢轮次。
- 留给预言家：不确定首刀牌身份，可保留解药保护拿到警徽的预言家（守-救-守模式）。
- **一晚只能用一瓶药**（规则见 `rules.md` §二.2）：解药用过当晚不能毒，反之亦然。

**毒药（最考验功力）**：
- **明确狼人时一击毙命**：2-3 轮发言抿出铁狼后再毒。
- **避免盲毒**：盲毒到好人概率远大于狼，毒到神职（尤其预言家）直接崩盘。
- **逼跳战术（高招）**：跳明女巫，宣布"今晚要毒某个范围（如警下三张可疑牌中选一）"，逼真神跳明身份，狼人也可能被迫穿神衣暴露。
- **银水补救**：解药救起的人表现异常（发言变形、归票狼队），可用毒药处理。

#### 反向锁定：从狼刀动作读狼面（女巫核心功力）

毒药决策最难的点不是"谁像狼"，而是"**狼队为什么这么刀**"。狼队的刀口选择会暴露他们的视角和恐惧——这是女巫最锋利的武器：

| 狼刀动作 | 反向锁定的狼面 | 解读 |
|---|---|---|
| **刀刚起跳的预言家** | 该预言家是真预；被刀预言家报的查杀/警徽流目标**狼面明显升高** | 狼怕真预验出队友，所以灭口 |
| **刀警长（预言家+警徽）** | 警长警徽流第一目标**狼面大幅抬高** | 狼确认警长下一夜会验到他 |
| **刀银水（被女巫救过的人）** | 银水基本是好人；狼在切好人战力 | 但也可能银水是自刀狼被识破 |
| **刀平民** | 狼在打屠民路线，狼藏在神坑里 | 女巫要重点保护剩余神 |
| **刀女巫自己** | 女巫是狼队优先目标，正常 | 但若女巫未暴露被刀，可能是颜杀/位置推测 |
| **空刀** | 狼在等女巫解药用完；或狼队优势想拖轮次 | 女巫要警惕下轮集中火力 |
| **刀非警徽流目标** | 狼队可能知道警徽流目标里有队友，避开 | 这种刀法暴露了"狼队知道谁是好人" |

**实战模板**（教练给女巫用户的分析话术）：

> "狼刀了 4 号警长——4 号之前宣告今晚验 8 号。**狼刀 4 这个动作本身，让 8 号狼面大幅升高**。如果不是 8 号怕被验出，狼队完全可以刀 7（银水警徽继承人）或刀你（女巫），没必要急着灭口预言家。**8 号是重点毒药候选**。"

> 注：以上"明显升高""大幅抬高"均为定性判断（狼人杀的概率无统计依据，不给虚假精确数字），实际撒毒决策仍要结合场上其他信息综合判断。

**反向锁定的三条铁律**：
1. **狼刀越急切，目标越重要**——刀刚起跳的预言家=怕真预；刀警徽流第一目标=怕被验。
2. **狼刀越反常，狼队越藏牌**——空刀、刀非焦点位，说明狼队有底牌在潜伏。
3. **狼刀和警徽流联动时，警徽流目标狼面飙升**——这是女巫撒毒的黄金信号。

### §2.3 盲毒判断标准

> 术语锚点：**盲毒** = 无证据乱毒。规则上毒药可指定任意玩家，这里讨论的是**决策标准**。

**核心立场：盲毒是高代价决策，默认不盲毒。** 仅在下面三种情况下才考虑。

**值得盲毒的极少数情况**：
1. **女巫被首刀 + 有颜杀把握**（不能自救，`rules.md` §五.2）：不毒等于浪费，毒还有概率带走狼。但**必须先报银水留遗言**，且仅在有把握时撒毒——无把握宁可不毒，把决策权交给警长/真预。
   - **颜杀边界（教练责任）**：教练看不到玩家表情/语气，**不主动评估颜杀**。仅当用户**明确告知颜杀判断**（如"我看 7 号表情像狼""7 号眼神躲闪"）时，教练才基于该用户提供的信息给盲毒建议；用户没主动说颜杀，教练默认按"无颜杀把握"处理（即不毒）。
2. **狼人绑票即将获胜**：不毒好人必输，毒还有翻盘可能。
3. **有明确怀疑对象**：经过 2-3 轮发言后对某张牌已形成"铁狼"判断。

**9 人局加倍禁忌**（§12.3）：
- 9 人局盲毒到好人概率 > 狼，**毒错直接 2 打 3，下一轮就输**。
- 9 人局女巫被首刀时：**首选不毒 + 留遗言**；盲毒仅作为"有铁证/颜杀把握"时的备选。

**通用盲毒禁忌**：
- 不要因为"赌一把"心理盲毒。
- 不要在没有信息的情况下首夜盲毒。
- 不要在被激怒/情绪化时盲毒。

> **教练口径**：用户问"该盲毒吗"时，**默认回答"不毒"+ 理由**。除非用户给出铁证或**明确告知颜杀判断**（见上方"颜杀边界"）才考虑毒。

### §2.4 女巫发言不暴露身份

- **首夜用了解药后**：以平民视角发言，不带"银水"信息特征。
- **不要主动保护银水**：过度维护被救起的人会暴露身份。
- **闭眼玩家口吻**：多用"我听着 X 像狼"、"我觉得 Y 发言有问题"等不确定语气。
- **避免"上帝视角"语气**：不要表现出"我知道刀口是谁"。

### §2.5 不同药量下的打法

| 状态 | 核心打法 |
|---|---|
| **双药在手** | 隐藏身份，观察银水，保留解药保护预言家，谨慎开毒。 |
| **单药（解药已用，毒在手）** | 可考虑起跳带队，给好人排掉两个坑位（自己+银水），压缩狼人生存空间；毒药威慑指定放逐目标。 |
| **单药（毒已用，解药在手）** | 极少见，需极度隐藏，把解药留给关键预言家或自己。 |
| **无药** | 已无技能价值，可光明起跳为好人更新信息，归票带队。 |

## §3 猎人

强自证身份的神，威慑力大于实际开枪，核心是"携枪威胁指定放逐"。

> 规则前提：猎人开枪触发条件、被毒/被白狼王带走失效——详见 `rules.md` §四.2。

### §3.1 何时开枪

- **被狼刀出局**：可开枪，带走认定的狼。
- **被投票出局**：可开枪，证明身份，带走最可疑的狼。
- **不能开枪**：被女巫毒死、被白狼王自爆带走、殉情（`rules.md` §四.2）。

**开枪目标优先级**：
1. 认定的铁狼（最佳）。
2. 高位认村民的可疑牌（消耗好人排水）。
3. **黑枪（不开枪）**：信息不足时宁可不带，避免盲带到神职。

**禁忌**：盲带是最差选择——带到好人概率远大于狼，带到女巫/预言家/守卫直接坑崩好人。

### §3.2 猎人被毒不能开枪

- 规则硬伤，无法规避（`rules.md` §四.2）。
- **前期不要过早暴露猎人身份**，避免成为狼人/女巫盲毒目标。
- 若已被毒：遗言中清晰交代自己的判断、点狼坑，为好人提供最后信息。

### §3.3 猎人发言节奏

- **第一天通常不跳**：由女巫带队，猎人观察局势。最好以"认村民"口吻发言避免被归票。
- **一旦被归票/被刀**：立刻翻牌开枪证明身份，不要黑枪。
- **第二天起**：信息量足够时可考虑跳明带队，利用枪的威慑归票狼人。
- **避免与女巫冲突**：猎人起跳会与女巫带队功能重叠，需配合而非拆台。

### §3.4 猎人抿人找狼

- 听发言的逻辑漏洞、心路历程饱满度。
- 观察票型：跟票狼队的牌高度可疑。
- 注意"为站队而站队"的牌——编不出饱满心路的多半是狼。
- 第二天已有信息量，基本不需要黑枪，带走认定的狼即可。

## §4 守卫

理论收益无上限（可创造多个平安夜），但**无法自证身份**，是狼悍跳目标。**在女巫出局前不要跳明身份**。

> 规则前提：空守/连守/同守同救（奶穿）规则——详见 `rules.md` §五.2。

### §4.1 第一夜守谁

| 策略 | 含义 | 优劣 |
|---|---|---|
| **空守**（主流） | 不守任何人，期望女巫盲救形成平安夜 | 降低"同守同救"事故率；守+预配合可打两天警徽流。整体收益高于自守。 |
| **自守** | 守自己 | 保证见到第一天太阳，但预言家中刀则后续全废；降低第二晚灵活性，容易被抿出。 |
| **守预言家** | 高风险高收益 | 预言家中刀+守卫守住+女巫不救=平安夜；但女巫同时救则奶穿双失效。 |

**建议**：玩家整体配置越高，越倾向空守；自刀率高的对局可考虑自守。

### §4.2 守-救-守模式（双神盾配合）

> 奶穿规则见 `rules.md` §五.2，本节只讲战术配合。

- **守-救-守模式**（有女巫板子）：第一晚空守 → 女巫救预言家 → 第二晚守预言家 → 第三晚女巫可再救 → 第四晚再守预言家。理想情况下预言家可活到第四个白天。
- **赌心态**：身份暴露后，狼人通常认为你会守预言家，此时可反其道自守，抢出一个轮次。

### §4.3 双神盾配合

- 守卫 + 女巫 + 预言家的"三神盾"是好人终极阵容。
- 标准安排：守卫守预言家、女巫解药留预言家、预言家打警徽流。
- 女巫光明起跳更新银水信息，呼吁守卫隐藏身份守护自己，同时开毒——一夜过后身份立明。
- **禁忌**：守卫和女巫同一晚保护同一人=奶穿。务必通过发言默契错开。

## §5 平民 / 白痴

> 规则前提：白痴翻牌机制（失投票权、只对公投生效、被刀被毒正常死）——详见 `rules.md` §四.2 + §五.2。

### §5.1 平民藏身份

**装神（高配）**：
- 警上可"诈身份"起跳预言家，听后置位反应（真预言家知道你是假的，悍跳狼和被查杀的不知道），通过表情判断狼人。**但必须退水**（`rules.md` §一.1：已退水不可再上警），不能真抢警徽。
- 表水为主：清晰交代思考过程，让好人感受到你的好人视角。
- 不要过度跳神：会脏真神的身份。

**白痴藏身份**：
- 白痴表面与平民无异，是狼人较难识别的隐神牌。
- 屠神局中避免暴露；屠民局中可为平民挡刀。
- 不要主动跳明白痴——技能是被动触发的，提前暴露等于送狼人信息。

### §5.2 平民为神职让路

- 不要和真神抢归票权。
- 警上诈身份后必须退水。
- 听到真神跳明后立刻配合站边。
- 必要时主动为神挡刀（认神衣服混淆狼人视线）。

### §5.3 抿狼、跟票技巧

- **抿狼**：通过发言漏洞、票型分析、心路历程饱满度判断。
- **跟票**：跟随坐定的好人（金水、女巫、预言家）投票，避免分票。
- **禁忌**：无理由弃票——弃票等于放弃排水，且容易被狼人利用形成绑票。

### §5.4 白痴被票后发言策略

- **翻牌免死**：被公投时亮明身份，免疫此次放逐，但**失去投票权**（规则见 `rules.md` §四.2）。
- 翻牌后可以**插嘴**，行使好人方最大的归票权（甚至可以改女巫的归票）。
- **担任警长被放逐**：必须移交或撕掉警徽，此后只有发言权。
- **翻牌后发言要点**：明确点出狼坑、归票指引好人、干扰狼人绑票节奏、残局扛推为好人争轮次。

## §6 狼人阵营

狼人团队四大角色分工：**悍跳狼、冲锋狼（煽动狼）、倒钩狼、深水狼（山洞狼）**。

> 规则前提：自爆类型、狼王/白狼王开枪条件——详见 `rules.md` §四。

### §6.1 悍跳狼标准套路

**核心目标**：跟真预言家抢警徽，扛推真预言家。

**发言模板**（完全模仿真预言家）：
1. 起跳："我是真预言家，昨晚验的 X 号，金水/查杀。"
2. 留警徽流：和真预言家一样的格式。
3. 聊心路历程：解释为什么验 X。
4. 点评前置位：攻击真预言家的发言漏洞。
5. 归票：拉警下票。

**报假查杀/假金水的几种玩法**：

| 操作 | 含义 | 适用场景 |
|---|---|---|
| **给真预言家发查杀** | 最常见的对冲手段 | 标准对跳，制造 1v1 对立 |
| **给狼队友发查杀（狼踩狼）** | 牺牲队友做高自己身份 | 多狼连坐、需要清洗狼面时 |
| **给狼队友发金水** | 队友可正大光明冲锋 | 高端花套路，缺点是把自己和队友打成焦点 |
| **给刀口发金水** | 制造"被验过的好人"假象 | 狼连坐中夹个好人 |
| **给警上后置位发金水** | 试探反应 | 听金水发言好可继续博力度 |

### §6.2 倒钩狼战术

**两种形态**：

**（1）阴阳倒钩（悍跳狼优势时）**：
- 主动跳一个神（如女巫、白痴）出来帮真预言家号票。
- 但发言**故意聊得不太好**，让好人觉得你像狼。
- 收益：脏真预言家身份（"真预言家团队里怎么有狼面？"），做高悍跳狼及金水身份。
- 真神可能跳出来站你这边，进一步做高你狼队友身份。

> 注：阴阳倒钩的核心是"跳女巫/白脏真预身份"，**不涉及自刀和警徽传递**。涉及自刀的"双狼悍跳+飞徽"是另一个独立的高级战术，见下方（3）。

**（2）深水狼（悍跳狼劣势时）**：
- 铁站边真预言家，抗推自己的悍跳狼队友。
- 把自己身份做高，免于被预言家查验。
- 决赛圈利用"铁好人"身份抗推其他好人，一举拿下胜利。
- 极端操作：开局夜间就不睁眼，只举手示意队友身份，以"天生闭眼"切入发言。

**（3）双狼悍跳 + 自刀飞徽（高端独立战术，与阴阳倒钩不同）**：
- **适用场景**：狼队有两只狼都愿意悍跳预言家（双跳），且倒钩狼愿意在后续以预言家身份接管警徽流。
- **逻辑链**（必须按顺序执行）：
  1. 首日白天警长竞选时，**狼 A 悍跳预言家**抢警徽（同时可能有真预言家对跳）。
  2. 狼 A 当选警长后，打正常的假警徽流（报假验人）。
  3. **第二夜狼队让狼 A 自刀**（狼 A 是狼阵营，可自刀；预言家是好人不能自刀）——狼 A 死前按预谋**飞徽给狼 B**（倒钩狼，引用 `rules.md` §三.4 警长死后飞徽机制）。
  4. 狼 B 接徽后**以预言家警长身份**打假警徽流，用假验人信息覆盖真预言家的真验人，把真预言家挤出好人信任的"验人信息源"。
- **与阴阳倒钩的区别**：阴阳倒钩跳女巫/白痴脏身份；本战术狼 B 跳的是**预言家**（接徽后接管警徽流），身份设定完全不同，不要混淆。
- **风险**：狼 B 需要在双跳场景下有足够饱满的假心路支撑"预言家视角"，否则容易被识破。

### §6.3 狼美人连人策略

> 规则前提：狼美人连人机制（单向同死、不能自刀、连两晚不能同一人）——详见 `rules.md` §五.1。

- **连人选择**：连那些大概率被公投的好人，制造"投狼美人=好人陪葬"的威胁。
- **配合悍跳狼**：狼美人连真预言家支持者，压缩好人投票空间。
- **应对**：好人需先毒/刀狼美人，或避开狼美人直接推其他狼。

### §6.4 狼人首夜刀人选择

| 刀法 | 目的 | 风险 |
|---|---|---|
| **刀神**（预/女/猎/守） | 削弱好人核心战力 | 高收益，但守卫可能守中、女巫可能救起 |
| **刀民** | 隐藏意图，避免同守同救事故 | 收益较低 |
| **刀警长候选** | 干扰警徽流向 | 容易暴露意图 |
| **自刀**（骗药） | 骗女巫解药，做高自己"银水好人"身份 | 女巫不救则血亏，深水狼自刀风险极大 |
| **精准首刀** | 直接刀中神（如守卫） | 高配狼队的标志，"落刀即神" |

### §6.5 狼人自爆战术

> 规则前提：自爆机制、双爆吞警徽、白狼王自爆带人——详见 `rules.md` §四 + §三.3。本节只讲战术时机。

**自爆时机选择**：
- 悍跳狼发言劣势、警徽无望时：自爆吞信息。
- 狼队轮次领先（如 3 狼对 2 神）：悍跳狼可直接自爆拍刀，依次刀神锁定胜局。
- 倒钩狼做高身份后：自爆坐实狼同伴身份。

**白狼王自爆带人的战术价值**：
- 自爆带走真预言家是最强破神手段（被带预言家无遗言无技能）。
- 常配合连爆吞警徽。
- **好人应对**：预判白狼王自爆目标，提前让真神跳明留信息。

### §6.6 狼人之间的隐藏配合

- **狼踩狼**：队友之间互相攻击，做高其中一人身份。需要战术统一，否则容易暴露。
- **互不发言**：狼队友之间避免直接互动，减少关联性。
- **隔空配合**：通过站位、投票、警徽流等间接信号配合，不直接对话。
- **统一口径**：避免冲锋狼和倒钩狼互相矛盾。倒钩狼可主动踩冲锋狼（"4 号太激进，像狼"），但逻辑必须自洽。
- **狼队分工**：头狼（领头）安排事务，其他狼在头狼未暴露时听指挥。

### §6.7 狼人冒充神职（狐假虎威）

- **冒充女巫**：跳明女巫，归票号票。风险：真女巫毒药在手可直接毒杀。
- **冒充守卫**：在女巫出局后跳明守卫。风险：真守卫无法自证，但可制造混乱。
- **冒充白痴**：被公投时翻牌（但狼人不能真翻牌，`rules.md` §四.2）。多用于威胁好人"投我就翻牌"。
- **冒充猎人**：携枪威胁。风险：真猎人可开枪带走你。
- **核心原则**：除非有特殊战术，不要随便声称角色——表演技巧（角色融合 + 切换视角）是关键。

## §7 警长 / 警下发言策略

> 规则前提：警长竞选流程、警长特权、双爆吞警徽——详见 `rules.md` §一.1 + §三。

### §7.1 警上发言：站位与拉票

**真预言家警上发言**：
1. 起跳 + 报查验
2. 留警徽流（带理由）
3. 聊心路历程（饱满是关键）
4. 点评前置位
5. 归票号召："希望大家把警徽投给我。"

**狼人悍跳警上发言**：
- 完全模仿真预言家模板
- 加入攻击真预言家的内容
- 制造"力度对比"压制真预言家

**其他牌警上发言**：
- **诈身份**：平民可警上跳预言家试反应，但必须退水。
- **冲锋狼**：用逻辑站边悍跳狼，攻击真预言家支持者。
- **倒钩狼**：伪装好人，主动站边真预言家，攻击狼队友。
- **深水狼**：全程划水，避免暴露逻辑漏洞。

### §7.2 警下发言：分析、抿人、投票引导

- **分析警上剧情**：两个对跳预言家的力度对比。
- **抿人**：观察警上各位牌的发言状态、表情。
- **投票引导**：明确站边某一方，号召其他警下牌跟随。
- **变票说明**：变票必须聊清心路历程——"为站队而站队"的狼人编不出饱满逻辑。

### §7.3 警长第一视角的特殊责任

- **1.5 票归票权**：警长投票算 1.5 票，是关键票（`rules.md` §三.1）。
- **决定发言顺序**：从死左还是死右开始发言，影响信息流向。
- **警徽移交**：警长出局时可移交警徽给任意玩家，新警长继承权利（`rules.md` §三.4）。
- **撕警徽**：也可撕掉警徽，本局无警长。
- **PK 台不能投票**：警长平票进入 PK 时，警长也不能投票（`rules.md` §三.5）。

## §10 发言技巧

### §10.1 节奏控制

- **先报信息式**：预言家、女巫、猎人等有信息量的牌，先报核心信息再分析。
- **先分析式**：平民、后置位发言的牌，先分析前置位发言再表态。
- **核心原则**：信息密度饱满、逻辑自洽、心路历程清晰。

### §10.2 站位话术

- **站 A 方**："我听完两边的发言，我站 A 方。理由是……"（给出具体理由）
- **站 B 方**："B 方的 X 号发言让我觉得更像真预言家，因为……"
- **两边不站**："我听着两边都有问题，我想听更多发言再决定。"（注意：两边不站容易被怀疑成隐狼）

### §10.3 压麦控场

- **慢节奏、长发言**：用饱满的分析压制场上节奏，让狼人没有插话空间。
- **重复关键信息**："我再强调一遍，X 号是查杀，Y 号是金水。"
- **明确归票**："今天必须把 X 号投出去，否则好人没机会。"
- **适合**：真预言家、坐定的女巫、白痴翻牌后。

### §10.4 抢麦破节奏

- **快速切入**：在前置位发言结束后立刻接话，打断狼人节奏。
- **强势归票**：直接宣布"我票 X 号"，先发制人。
- **适合**：冲锋狼、需要打破对方节奏时。

### §10.5 避免被打成焦点

- 不要过度跳神、过度归票。
- 警上诈身份后必须退水。
- 不要和真神抢风头。
- 表水为主，让好人感受到你的好人视角。
- 不要无理由弃票、不要分票。

### §10.6 装弱、藏强、虚张声势

- **装弱**：藏神身份时以平民口吻发言，避免暴露夜间视角。
- **藏强**：守卫、猎人前期不跳明，潜伏观察。
- **虚张声势**：狼人冒充神职携枪/携毒威胁，需要表演技巧。

## §11 投票策略

> 规则前提：投票形式、票值、平票 PK 流程——详见 `rules.md` §三.5。

### §11.1 投票时机

- **警长竞选投票**：警下牌统一举票，允许弃票，跟票无效。
- **放逐投票**：所有玩家同时举票，允许弃票，跟票无效。
- **PK 台投票**：平票玩家进入 PK 发言后，非 PK 玩家投票。

**第一轮谁先投**：通常由坐定的好人（金水、女巫）先表态，其他好人跟随。

### §11.2 票型分析

**狼人通常会投谁**：
- 冲锋狼：冲票给悍跳狼队友。
- 倒钩狼：上票给真预言家（做高自己身份）。
- 深水狼：跟票多数，隐藏在人群中。

**票型分析要点**：
- **冲票给悍跳狼的牌**：里面可能有站错边的民 + 冲锋狼。
- **上票给真预言家的牌**：通常是好人 + 倒钩狼。
- **弃票的牌**：可能是犹豫的好人，也可能是隐狼。
- **变票的牌**（第二次投票改变站边）：**最值得关注**——变票背后必然有原因和逻辑。好人能聊清变票原因，狼人编不出饱满心路历程。
- **核心金句**："发言会骗人，票型可能是假的，逻辑也可能是反的，但行为不会。"

### §11.3 平票 PK 处理

> 流程规则见 `rules.md` §三.5。本节讲 PK 发言策略。

**PK 发言策略（真预言家）**：
- 根据第一次投票票型修改警徽流。
- 把第一警徽流给到悍跳狼团队的牌。
- 对给自己上票的牌鼓励 + 适当压力（防倒钩狼）。
- 对冲票给悍跳狼的牌互动："站错边没关系，现在站回来就行，否则会被怀疑成冲锋狼。"
- 游说弃票的牌勇敢上票。
- 攻击悍跳狼两轮发言的漏洞。

### §11.4 弃票的利弊

**利**：犹豫不决时避免投错人；战术性弃票制造特定票型。

**弊**：放弃排水机会；容易被狼人利用形成绑票；容易被怀疑成隐狼。

**结论**：除非有明确战术目的，否则不要弃票。跟票坐定好人是更安全的选择。

### §11.5 "测试性投票"的风险（高阶陷阱）

**定义**：在两个对跳预言家难辨真伪时，提议"先票其中一个的查杀目标，看身份定真伪"。听起来像聪明策略，实际是**好人最常见的自毁操作之一**。

**典型场景**：
- 3 号跳预言家报 6 号查杀，4 号跳预言家报 7 号查杀
- 好人困惑，有人提议："先票 6 号看看，6 号是狼说明 3 号真预，6 号是民说明 3 号假"

**致命风险**：**狼队会顺水推舟，让假查杀被坐实**

| 好人想法 | 狼队实际操作 | 结果 |
|---|---|---|
| "票 6 号测试 3 号真伪" | 警长 3 号（狼）归票 6 号，狼队冲锋票 6 号 | 6 号（平民）出局 |
| "6 号死了看身份就知道" | 法官宣布 6 号是平民 | 3 号辩解"我验错了但我还是真预" |
| "那 3 号有问题" | 3 号继续用话术 + 警长身份压制 | 好人已经少 1 民，且 3 号仍是警长 |

**为什么测试性投票帮倒忙**：
1. **轮次亏损**：好人花一轮票平民，等于白送狼队一轮
2. **假查杀被坐实**：狼队的假查杀"验过了"（虽然验错），反而让悍跳狼有话说
3. **信息未必有用**：6 号死后法官只宣布"平民"，不直接证明 3 号是狼——3 号仍可辩解
4. **被狼队利用**：警长归票权+冲锋狼冲票，轻松让测试目标出局

**正确策略**：
- **两个对跳预言家时，直接票其中一个预言家**，不要票他们的查杀目标
- 票预言家 = 直接解决对跳，验明正身
- 票查杀目标 = 绕弯路，且被狼队利用

**教练提示模板**（用户提议"测试性投票"时）：
> "⚠️ 别票 6 号测试。**狼队会让你测试成功**——警长 3 号归票 6 号，6 号（平民）死后 3 号辩解'验错了但还是真预'，你白送一轮。**正确做法是直接票 3 号或 4 号其中一个预言家**，解决对跳本身。如果你拿不准 3 和 4 谁真，听心路饱满度+警徽流走向+8 号这种'站边牌'的行为细节，而不是票他们的查杀目标去绕弯。"

**例外（通常不建议，但需视局势判断）**：
- **铁证场景**：如果悍跳狼已经铁证如山（如双死、自爆吞警徽等铁证），可以票他的查杀目标排水。但此时已经不需要"测试"了。
- **双狼悍跳场景**：若两个对跳预言家**都是狼**（罕见但存在），票查杀目标（大概率好人）反而能排水暴露狼队结构。但**判断"双狼悍跳"本身需要铁证**（如后续位置出现第三跳、票型异常集中等），无铁证时仍按"票预言家本身"处理。

## §12 9 人预女猎专项（节奏快、单神即胜负）

9 人局：3 神（预女猎）+ 3 民 + 3 狼。**节奏极快、容错极低、单神死亡常常直接定胜负**。

> 规则前提：9 人局配置、女巫首夜看刀口、盲毒禁忌——详见 `rules.md` §二.2 + §六。

### §12.1 9 人局的核心特征

| 维度 | 9 人局特点 | 教练注意 |
|---|---|---|
| **节奏** | 3-4 轮内分胜负 | 策略要"短平快"，不要给长线布局 |
| **信息密度** | 神少、信息源少 | 预言家死=好人基本崩盘 |
| **狼人战术** | 悍跳成本高（狼少自爆亏） | 倒钩、深水比悍跳更常见 |
| **女巫重要性** | 极高（单女巫单猎人撑全场） | 女巫毒错=直接输 |
| **猎人威慑** | 极高（3 神中 1 神） | 猎人前期藏、被推时翻牌带狼 |
| **平民责任** | 极重（3 民死光直接输） | 平民要敢站边、敢投票、敢抿狼 |

### §12.2 9 人局预言家（最脆的神）

- **首夜必救人**：女巫首夜不救=预言家大概率死=好人崩盘。这是 9 人局铁律。
- **警徽流要短**：9 人局轮次少，警徽流留 1-2 天即可，不要留 3 天（用不上）。
- **藏不住**：9 人局信息少，预言家几乎必须起跳，否则好人没有信息源。
- **被悍跳时**：饱满心路 > 报查杀。9 人局狼少，悍跳狼的"睁眼视角"更容易暴露。
- **死亡预案**：随时准备好"今晚死了警徽飞谁"，每次发言结束都更新警徽流。

### §12.3 9 人局女巫（最关键的胜负手）

- **首夜标准救人**：除非刀口是自己（不能自救），否则必救。
- **毒药要慎之又慎**：9 人局毒错一个好人=直接 2 打 3，下一轮就输。
- **盲毒禁忌加倍**：9 人局盲毒到好人概率 > 狼，按 `strategy.md` §2.3 的颜杀边界判断——用户未明确告知颜杀判断时，默认不毒。
- **第二夜考虑起跳带队**：9 人局信息少，女巫跳明可以给好人排 2 个坑（自己+银水），价值极大。
- **银水观察要细**：被救的人第二天发言有异常（变形、归票狼队），立刻毒掉。

### §12.4 9 人局猎人（最强威慑）

- **第一天不跳**：以认村民口吻发言，藏好身份。
- **第二天观察**：听两轮发言，识别铁狼。
- **开枪条件**：被推/被刀时立刻翻牌，带走认定的铁狼。**不要黑枪**。
- **被毒应对**：9 人局猎人被毒=好人少一半火力。前期不要暴露，避免被盲毒。

### §12.5 9 人局平民（责任最重）

- **不要诈身份**：9 人局信息少，诈身份容易脏真神的牌，得不偿失。
- **敢站边**：平民要敢于明确站边真预言家，给女巫、猎人信心。
- **跟票**：跟随坐定好人投票，不要分票、不要弃票。
- **抿狼**：听发言饱满度，9 人局狼少，狼人编不出 3 张好牌的逻辑。

### §12.6 9 人局狼人（倒钩优先）

- **悍跳慎用**：3 狼中悍跳 1 个，剩 2 狼。悍跳狼被识破=狼队少 1/3 战力。
- **倒钩是主流**：深水站边真预言家，做高自己身份，决赛圈翻盘。
- **首刀选择**：
  • 刀预言家（高收益但女巫大概率救）
  • 刀女巫（极高风险，但成功=赢一半）
  • 刀平民（稳健，屠民路线）
  • **不推荐自刀**：9 人局狼少，自刀亏一个轮次基本输。
- **不要自爆**：9 人局自爆成本极高（少一狼），几乎只在残局拍刀时用。

### §12.7 9 人局常见残局

| 残局 | 处理 |
|---|---|
| **3 人残局**（2 好 1 狼 / 1 好 2 狼） | 投票定胜负。好人要识破狼的伪装。 |
| **4 人残局**（2 好 2 狼 / 3 好 1 狼） | 狼可能绑票，好人要拆票或弃票制造平票。 |
| **女巫+猎人双神在** | 几乎稳赢，女巫毒狼+猎人开枪带狼。 |
| **预言家死+猎人死** | 好人靠平民硬推，难度极大。 |

#### 9 人局孤狼残局识别（倒钩狼深水后的最后挣扎）

9 人局狼少，打到 1-2 轮后经常出现"孤狼 1 打 5"的残局。**孤狼最可能的形态是倒钩狼**——前面铁站真预、票过狼队友、身份做高。识别和应对方法：

**孤狼的常见伪装**：
- "我是铁好人，票过 3 号、站边 4 号、跟票 7 号警长。"
- "X 号（攻击目标）发言全程没贡献，建议警长归 X。"
- "狼坑只剩 1 狼，我排掉我自己，你们剩下的人里推。"

**孤狼暴露的 3 个致命信号**（好人抓住即终结）：

| 信号 | 解释 | 应对 |
|---|---|---|
| **事后诸葛亮** | "我早看 X 像狼"但前几轮没提 | 警长追问："你昨天发言有提到吗？" |
| **主动归票** | 深水狼本应跟票，他却抢着归票 | 警长反推："你为什么这么积极？归票该我来。" |
| **攻击平民逻辑过于精准** | 倒钩狼踩好人会踩在"恰好不伤核心"的点上 | 警长听其心路饱满度——狼人逻辑精准但缺细节 |

**警长应对孤狼的标准操作**：
1. **不要轻信铁站边的好人**——9 人局倒钩狼是主流战术
2. **追问历史发言**："你说早看 X 像狼，昨天第 N 轮你原话是什么？"
3. **观察归票积极性**：铁好人归票慎重，孤狼归票积极
4. **孤狼没视野盲区**：他能精准点评每个人，因为他是睁眼视角

**孤狼翻盘的唯一路径**（教练预判狼队战术）：
- 刀女巫（带走毒药威胁）→ 决赛圈扛推 1 平民 → 赌猎人黑枪
- 翻盘概率 < 5%，好人只要不内讧几乎稳赢

**教练给好人的提示**：
> "场上只剩 1 狼。最可能的藏身位置是**之前铁站真预、票过狼队友的牌**（典型倒钩狼面）。**不要因为他站对了边就排除他**——倒钩狼的核心伪装就是'站对边'。建议警长追问他的历史发言一致性，重点看他是否'事后诸葛亮'或'主动归票'。"

## §13 12 人预女猎白专项（白痴翻牌带队、残局强）

12 人预女猎白：4 神（预+女+猎+白）+ 4 民 + 4 狼。**与预女猎守的最大差异：缺守卫，多白痴**。

> 规则前提：白痴翻牌机制（失投票权、只对公投生效）——详见 `rules.md` §四.2 + §五.2。

### §13.1 与预女猎守的核心差异

| 维度 | 预女猎守 | 预女猎白 |
|---|---|---|
| **预言家生存** | 守卫可保 3-4 轮 | **无守卫，预言家极脆**，首夜必死概率高 |
| **首夜救人** | 守卫空守+女巫救=平安夜 | **女巫必须救**，否则预言家几乎必死 |
| **信息神** | 守卫也算半个信息神 | 只有预言家一个信息神，更稀缺 |
| **残局核心** | 守卫+女巫+预言家三神盾 | **白痴翻牌带队**是残局核心 |
| **白痴角色** | 无 | 翻牌不死、归票权最大、扛推能力强 |
| **狼人倒钩空间** | 较小（守卫能反制） | **大**（缺信息神，倒钩狼更难被识破） |
| **屠神 vs 屠民** | 屠神难（守卫难杀） | **屠神相对容易**（白痴翻牌前算神） |

### §13.2 预女猎白·预言家（更激进的打法）

- **首夜必跳**：缺守卫，预言家不跳=信息源死=好人崩盘。
- **警徽流必须饱满**：留 2-3 天，因为随时可能死。
- **不要藏**：12 人预女猎白信息神只有你一个，藏=浪费。
- **死亡前抢报**：每次发言结束都更新警徽流和后续安排。
- **被双跳对冲时**：饱满心路 + 攻击悍跳狼"睁眼视角"。

### §13.3 预女猎白·女巫（更激进）

- **首夜必救**：12 人预女猎白预言家太脆，首夜不救=几乎必死。
- **第二夜考虑起跳带队**：女巫跳明可以替预言家分担火力，并给好人排 2 坑。
- **毒药保留到中后期**：缺守卫的局面下，毒药是好人最后的反制手段。
- **银水观察**：被救起的人发言异常立刻毒掉，不要犹豫。

### §13.4 预女猎白·猎人（更显性）

- **预女猎白猎人更显性**：因为缺守卫，女巫救药用完后，猎人可能成为最后一个保命神。
- **第二天起可以跳明带队**：信息神少，猎人跳明可以归票狼人。
- **开枪策略**：被推/被刀时带走铁狼，不要黑枪。

### §13.5 预女猎白·白痴（残局 MVP）

- **前期藏**：白痴表面与平民无异，前期不要暴露。
- **被票时翻牌**：被公投时亮明身份免疫放逐（但失去投票权）。
- **翻牌后归票权最大**：甚至可以改女巫的归票。
- **担任警长被放逐**：必须移交或撕警徽，此后只有发言权。
- **翻牌后发言要点**：
  • 明确点出狼坑（你的好人视角最清晰）
  • 归票指引好人
  • 干扰狼人绑票节奏
  • 残局扛推为好人争轮次
- **挡刀价值**：屠民局中可为平民挡刀（认神衣服混淆狼人）。

### §13.6 预女猎白·平民

- **不要过度诈身份**：白痴板缺信息神，诈身份容易脏真神。
- **跟票坐定好人**：跟金水/女巫/预言家投票。
- **为白痴让路**：听到白痴翻牌后立刻配合归票。

### §13.7 预女猎白·狼人（倒钩空间大）

- **倒钩空间大**：缺守卫+缺信息神，倒钩狼更难被识破。
- **首刀预言家**：预女猎白预言家最脆，首刀成功率高（只要女巫没救）。
- **针对白痴**：白痴翻牌前是神，狼人可以尝试刀白痴（识别较难）。
- **屠神路线可行**：白痴翻牌前算神，屠神胜利相对容易。
- **慎用自爆**：白痴板自爆吞警徽有效，但白痴翻牌后仍可带队，效果打折。

### §13.8 预女猎白常见残局

| 残局 | 处理 |
|---|---|
| **白痴+女巫双神在** | 几乎稳赢，白痴翻牌归票+女巫毒狼 |
| **白痴+猎人双神在** | 白痴翻牌归票推狼，猎人开枪兜底 |
| **白痴单神在** | 白痴翻牌归票扛推，平民配合 |
| **无神只剩民** | 难度极大，靠平民硬推 |

## §14 残局决策树（教练快速查阅）

### §14.1 3 人残局（2 好 1 狼 / 1 神 1 民 1 狼）

```
你是好人（神/民）：
├── 已知狼身份：直接投票出局
├── 1 神疑狼 + 1 民疑狼：
│   ├── 女巫在 → 毒疑狼
│   ├── 猎人在 → 票疑狼，被反票翻牌带狼
│   └── 平民在 → 投票定胜负，要敢投
└── 投票失败 → 进黑夜，狼刀定胜负

你是狼人：
├── 2 好人互疑：装好人，让好人互投
├── 1 好人明显怀疑你：自爆（残局自爆=拍刀胜）
└── 双好人都坐定你：刀关键神，赌最后轮
```

### §14.2 4 人残局

```
2 好 2 狼：
├── 好人：必须拆票或弃票制造平票（防狼绑票）
├── 狼人：尝试绑票（2 票即可出人）
└── 关键：好人不能让狼形成 2 票

3 好 1 狼：
├── 好人：投票出局狼，几乎稳赢
├── 狼人：深水装好人，扛推其他好人
└── 风险：好人互投让狼翻盘

1 神 3 狼：
├── 神只能赌一搏（女巫盲毒/猎人黑枪）
└── 几乎必输，但不要弃局
```

### §14.3 5+ 人残局

按"轮次账"算：
- 好人剩余命数 = 剩余神 + 剩余民
- 狼人剩余命数 = 剩余狼
- 谁的轮次多谁优
- 女巫毒药/猎人枪是关键翻盘点

### §14.4 轮次精算（教练必须掌握）

**轮次账公式**：
```
好人轮次 = 剩余神数 + 剩余民数 - 当夜预期死亡数
狼人轮次 = 剩余狼数
净轮次 = 好人轮次 - 狼人轮次
```

> **"当夜预期死亡数"边界**：默认按 1 算（狼刀通常杀 1 人），但**平安夜场景**（守卫守住 / 女巫救人 / 狼空刀）下为 0；女巫撒毒且毒中好人时为 2（狼刀+毒）。教练需结合当夜实际死亡情况调整，不能机械套 "-1"。

**净轮次判断**：
| 净轮次 | 局势 | 教练基调 |
|---|---|---|
| ≥ +3 | 好人碾压 | 稳健防守，不冒险 |
| +1 ~ +2 | 好人优势 | 继续推狼，注意防倒钩 |
| 0 | 焦灼 | 谁先犯错谁输 |
| -1 ~ -2 | 狼人优势 | 女巫毒药/猎人枪是唯一翻盘点 |
| ≤ -3 | 狼人碾压 | 好人靠对手失误 |

**关键轮次节点**：
- **好人净轮次 = 1 时**：必须当天票出 1 狼，否则进黑夜被刀 1 人后净轮次归零
- **好人净轮次 = 0 时**：女巫必须撒毒或猎人必须开枪，否则进黑夜必输
- **狼人净轮次 = 1 时**（狼多好人少）：狼队可以拍刀（自爆+刀神），好人几乎无解

### §14.5 孤狼残局识别清单（9/12 人局通用）

**孤狼最常见的伪装形态**：倒钩狼深水后做高身份，决赛圈扛推好人。

**识别孤狼的 5 个行为信号**（按重要性排序）：

| # | 信号 | 抓法 |
|---|---|---|
| 1 | **事后诸葛亮** | 追问历史发言："你说早看 X 像狼，第 N 轮原话是什么？" |
| 2 | **主动归票** | 铁站边的好人突然归票=想扛推别人=视角暴露 |
| 3 | **攻击逻辑过于精准** | 倒钩狼踩好人踩在"恰好不伤核心"的点，太精准=睁眼 |
| 4 | **从不被归票、从不当焦点** | 全程划水的"铁好人"，决赛圈往往是最后一只狼 |
| 5 | **抿神过度积极** | 倒钩狼需要为狼队找神，会过度试探"谁是神" |

**警长/好人应对孤狼的标准流程**：
1. 列出"铁站真预、票过狼队友"的牌 → 这些是孤狼候选
2. 追问每张候选牌的历史发言一致性
3. 观察归票积极性（铁好人慎重，孤狼积极）
4. 听心路饱满度（狼人逻辑精准但缺细节，好人细节丰富但可能犹豫）
5. **视轮次决定推还是放**：9 人局剩 1 狼且好人净轮次 ≥ 2 时，推错 1 平民仍可挽回；但**净轮次 = 1 时推错直接输**，此时必须谨慎，宁可多听一轮发言也不要盲推。判断不清轮次时，默认保守（不推）。

### §14.6 残局决策速查（按用户身份）

**你是预言家（已死）**：靠警徽流和遗言为好人留路线，新警长执行。
**你是女巫（毒药在手）**：算轮次——净轮次 ≤1 时必须撒毒，毒铁狼不毒疑狼。
**你是猎人（枪在手）**：被推/被刀时翻牌带铁狼，不要黑枪。
**你是平民**：跟警长票，不要分票，不要弃票。
**你是警长（任何身份）**：归票前追问历史发言，重点查倒钩狼信号。

## 教练结语

狼人杀是**没有纯策略纳什均衡的博弈**——任何套路都有克制之法，灵活应变才是高手本色。本手册的策略都是"可选策略"而非"唯一最优解"。教练给出建议时，**永远结合用户当前局的具体信息**，而不是套模板。
