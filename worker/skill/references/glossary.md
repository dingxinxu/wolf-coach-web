# 狼人杀术语速查词典

> 教练用法：用户用了不熟的术语、或你需要用术语给建议时查阅。

---

## 一、角色与状态类

| 术语 | 释义 |
|---|---|
| **金水** | 预言家验出的好人（"发金水"=报某人是好人） |
| **查杀** | 预言家验出的狼人（"报查杀"=指出某人是狼） |
| **银水** | 女巫解药救过的人（视为好人倾向） |
| **盲毒** | 女巫无证据乱毒人 |
| **盲守** | 守卫无信息首夜乱守 |
| **同守** | 守卫连续两晚守同一人（违规/无效） |
| **同守同救 / 奶穿** | 守卫与女巫解药同夜作用于同一人，目标仍死亡 |
| **双死** | 一夜死两人（如狼刀 + 女巫毒） |
| **底牌** | 玩家的真实身份牌 |
| **穿衣服** | 冒充某身份发言（民穿神衣、狼穿神衣） |
| **反水立警** | 被发金水后不接水，自己跳预言家 |
| **接水** | 接受预言家发的金水身份 |
| **银水票** | 银水玩家被怀疑时的解释性投票 |

## 二、战术与发言类

| 术语 | 释义 |
|---|---|
| **悍跳** | 狼人冒充神职起跳（最常见悍跳预言家） |
| **对跳** | 两人声称同一身份（如两个预言家对跳） |
| **倒钩** | 狼人深水装好人，站边真预言家迷惑好人 |
| **阴阳倒钩** | 狼故意站边真假预言家两边，制造混乱 |
| **深水狼 / 山洞狼** | 以平民身份潜伏、几乎不暴露的狼 |
| **冲锋狼 / 煽动狼** | 用逻辑站边悍跳狼、带节奏推神的狼 |
| **抿 / 抿身份** | 通过发言/表情猜测身份 |
| **颜杀** | 通过表情判断狼人 |
| **认神 / 跳神** | 公开声称自己是某神职 |
| **藏神** | 神职藏身份不暴露 |
| **煽风点火** | 故意挑拨、带节奏制造混乱 |
| **白嫖** | 不付出代价获取信息（如白嫖预言家验人） |
| **狼踩狼** | 狼人白天故意踩同伙做身份 |
| **生推** | 无神职信息时硬推狼坑 |
| **表水** | 平民清晰交代思考过程做好身份 |
| **诈身份** | 警上跳预言家试反应，听后置位发言判断狼人（必须退水） |
| **金刚狼** | 通过战术做高到极难被识破的狼 |
| **拍刀** | 狼人轮次领先时直接自爆+刀神锁定胜局 |

## 三、警长与流程类

| 术语 | 释义 |
|---|---|
| **警上** | 参与警长竞选的玩家 |
| **警下** | 未参与竞选、可投票选警的玩家 |
| **警徽流（镜子）** | 警长（多为预言家）报出的验人预告及死后传徽规则 |
| **撕徽 / 撕警徽** | 警长主动撕掉警徽传递暗号 |
| **飞警徽** | 警长死后将警徽传给指定玩家 |
| **双爆吞警徽** | 出现两次狼自爆，警长竞选作废，整局无警长 |
| **退水** | 警上竞选者退出竞选 |
| **压麦** | 发言超时占用麦克风（也可指故意长发言控场） |
| **抢麦** | 抢先发言，不按顺序 |
| **归票 / 号票 / 拉票** | 号召大家一起投票某人 |
| **绑票 / 冲票** | 集体投票试图放逐某人 |
| **分票** | 没有统一投票，分散投多人 |
| **单票出局** | 只有一人投票其他人弃票，被投者出局 |
| **PK** | 平票后两人再发言对决 |
| **弃票** | 投票时选择弃权 |
| **自爆** | 狼白天主动暴露身份死亡，跳过当天投票直接进夜 |
| **遗言** | 出局玩家最后的发言（通常仅首日死者有） |
| **两手 / 左右手** | 警长决定从死亡玩家左边或右边开始发言，或两边各起一个 |
| **警推** | 警长带票推人 |

## 四、局势与结算类

| 术语 | 释义 |
|---|---|
| **屠边** | 狼杀光全部神或全部民即胜 |
| **屠城** | 狼杀光所有好人才胜 |
| **平安夜** | 当夜无人死亡 |
| **空刀** | 狼人当夜不刀人（少见，多为战术） |
| **神坑 / 狼坑** | 神职或狼人嫌疑位的玩家池 |
| **倒牌** | 出局 |
| **水民** | 普通平民 |
| **归边** | 局势明显偏向某一阵营 |
| **轮次** | 一"轮次"=一个白天+一个黑夜；好人/狼人的轮次差是胜负核心 |
| **抢轮次** | 通过救人/守人/毒狼为好人争取额外轮次 |
| **第三阵营** | 丘比特情侣连线单独成阵，活到最后胜出 |
| **面杀 / 网杀** | 线下面杀 / 线上语音杀 |

## 五、关键短语（教练台词可直接用）

| 短语 | 用法 |
|---|---|
| "X 号是金水/查杀" | 预言家报验人结果 |
| "警徽流：验 A 飞 A，验 B 撕" | 警徽流标准报法 |
| "X 号请接水" | 让被发金水的玩家承认好人身份 |
| "我站 A 方" | 表态站边某一阵营 |
| "X 号发言像狼/像神" | 抿人结论 |
| "今天必须票 X 号" | 强势归票 |
| "黑枪" | 猎人不开枪（信息不足） |
| "盲毒" | 女巫无证据毒人（教练应强烈劝阻） |
| "奶穿" | 同守同救双失效 |
| "X 号穿衣服" | X 号在冒充神职 |
