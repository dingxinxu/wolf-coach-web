## 这是什么

`wolf-game-coach` 是一个**实时陪伴型**狼人杀教练技能。在你打狼人杀（线上语音局或线下面杀）时，每轮告诉你：

- 🎯 **该说什么**：可直接念的发言台词（≤120 字，符合实战发言时长）
- 🗳️ **该投谁**：明确的投票决策（首选+备选+理由）
- 🔍 **谁是狼**：每个玩家的身份概率推断（带理由）
- ⚠️ **风险在哪**：当前轮次的最大风险和规避方法
- 💊 **资源怎么用**：女巫毒谁、猎人开枪条件、守卫守谁、预言家验谁

**与传统攻略的区别**：

| 维度 | 传统攻略 | wolf-game-coach |
|---|---|---|
| **颗粒度** | "预言家要饱满起跳" | "你 4 号预言家验 7 号金水，警徽流留 8、6，台词：'我是真预...'" |
| **情境绑定** | 通用规则 | 基于你当前板子、身份、座位、轮次定制 |
| **可执行性** | 理论 | 可直接念的台词 + 明确投票决策 |
| **动态更新** | 静态 | 每轮根据新信息更新分析 |

## ✨ 核心功能

### 1. 🗣️ 可念台词（核心价值）

每轮给出**符合实战发言时长**的台词（警上 ≤120 字 / 普通发言 ≤100 字 / PK·遗言 ≤80 字），可以直接念出来。

**示例（预言家警上发言，105 字）**：
> "我是真预，昨晚验 7 号老陈金水，请 7 号接水。3 号大壮悍跳，他给 1 号发金水我严重怀疑是狼队互抬。警徽流先 8 再 6，验警下没发言的牌排狼坑。今晚若倒下：验 8 金水飞 7，查杀飞 6。希望警下把警徽投我，今天票 3 号。"

### 2. 🗳️ 智能投票决策

每轮给出**首选+备选**投票方案，带概率推理：

```
首选：投 3 号（悍跳狼），把警徽拿到手
备选：若票型 5>3 → 投 5（你的查杀）；若进 PK → PK 发言重点打 3 的"反咬视角"
```

### 3. 🔍 倒钩狼识别（高阶能力）

基于实战行为信号的**倒钩狼识别清单**，识破最深度的伪装狼：

- 事后诸葛亮（"我早看 X 像狼"但昨天没提）
- 主动归票拉仇恨（深水狼本应跟票）
- 攻击逻辑过于精准（睁眼视角）
- 站边时机过早且无悬念
- 从不当焦点、从不被归票

### 4. 💊 女巫反向锁定（独家能力）

**从狼队刀法反推狼面**——这是女巫毒药决策的核心功力：

> "狼刀了 4 号警长——4 号之前宣告今晚验 8 号。**狼刀 4 这个动作本身，把 8 号狼面从 75% 抬到 85%+**。如果不是 8 号怕被验出，狼队完全可以刀 7（银水）或刀你（女巫），没必要急着灭口预言家。**8 号基本可以下毒**。"

### 5. 🎯 残局精算

进入残局时自动算轮次账、识别孤狼、给出按身份的决策树：

```
3 好 vs 1 狼：净轮次 +1，今天必须投对
孤狼最可能藏在"票过狼队友+铁站真预边"的牌里
建议警长追问历史发言一致性
```

### 6. 🛡️ 防泄露保护

如果用户身份没暴露，**台词绝对不会泄露夜间视角**。例如女巫未报身份时，发言建议不会带"我刚才救了人"。

狼人用户的台词**绝不出现队友座位号**——保护狼团队不被关联。

## 🚀 快速上手

### 安装

技能已放在工程目录下，**无需额外安装**。如果你在另一个项目使用，复制整个目录：

```bash
# 复制到你的项目
cp -r .agents/skills/wolf-game-coach /path/to/your/project/.agents/skills/

# 或复制到用户级目录（全局可用）
cp -r .agents/skills/wolf-game-coach ~/.agents/skills/
```

ZCode/Claude Code 会自动发现 `.agents/skills/` 或 `~/.agents/skills/` 下的技能。

### 触发方式（任一即可）

| 触发方式 | 示例 |
|---|---|
| **自然语言** | "帮我打狼人杀"、"我抽到预言家"、"今晚有人开局" |
| **斜杠命令** | `/wolf-game-coach` 或 `/狼人杀` |
| **情境触发** | "9 人局，我是女巫，第一夜被首刀，要盲毒吗？" |

### 第一次使用

直接在 ZCode/Claude Code 对话里说：

```
帮我打狼人杀，12 人预女猎守，我抽到预言家，座位 7 号。
```

教练会自动进入交互模式。

## 🎮 完整使用流程

教练按下面 4 步状态机陪你打完整局：

```
┌──────────────────────────────────────────────────┐
│ 第 1 步：开局采集（SETUP）                          │
│   教练问：几人局？板子？身份？座位？熟人？            │
│   你答：12 人预女猎守，预言家，7 号                  │
└──────────────────────────────────────────────────┘
                       ↓
┌──────────────────────────────────────────────────┐
│ 第 2 步：夜晚报告（NIGHT REPORT）                   │
│   教练问：昨晚谁死了？你做了什么？                    │
│   你答：5 号被刀，我验了 8 号查杀                   │
└──────────────────────────────────────────────────┘
                       ↓
┌──────────────────────────────────────────────────┐
│ 第 3 步：白天发言（DAY SPEECH）                     │
│   你报告：所有玩家发言                              │
│   结尾说："本轮发言全部结束"                        │
└──────────────────────────────────────────────────┘
                       ↓
┌──────────────────────────────────────────────────┐
│ 第 4 步：投票 + 分析（VOTE + ANALYSIS）             │
│   教练问：票型？你答完后，教练输出 6 段分析           │
└──────────────────────────────────────────────────┘
                       ↓
                   回到第 2 步（下一夜）
                       ↓
                   游戏结束 → 复盘
```

### 每轮分析的 6 段格式

教练每轮严格按下面 6 段输出（精简模式）：

```
═══════════ 第 N 轮 ═══════════

【⚠️ 先回答这几个】（1-3 个关键追问，信息够就省略）
1. ...
2. ...

【态势】（一句话 + 紧凑表）
好人微优，你被狼队压制但警徽在手。

| 我=预言家 | 4 号 | 存活 | 警长 | 6 人存活 | 验人+警徽流 |

玩家档案：
| # | 动作 | 判断 | 一句话理由 |
| 3 | 悍跳 | 狼 75% | 警徽流逻辑漏洞 |

【发言】（核心）
▶ 可念台词（≤100 字）：
"..."

要点：① ... ② ... ③ ...

【决策】（投票/毒药/开枪，按状态选模板）
首选：投 X，因为 ...
备选：若 PK → 投 Y

【风险】（≤2 条）
- ...

【下轮等】
天黑后告诉我：① 谁死 ② 你做了什么。
```

## 📋 支持的板子配置

### 主流板子（深度专项支持）

| 板子 | 总人数 | 配置 | 特点 |
|---|---|---|---|
| **9 人预女猎** | 9 | 预+女+猎 / 3 民 / 3 狼 | 节奏快、单神即胜负、倒钩优先 |
| **12 人预女猎守** | 12 | 预+女+猎+守 / 4 民 / 4 狼 | 竞技性最强、双神盾配合 |
| **12 人预女猎白** | 12 | 预+女+猎+白 / 4 民 / 4 狼 | 白痴翻牌带队、残局强 |

### 扩展板子（基础支持）

狼王局、白狼王局、狼美人局、守墓人石像鬼板、6/8/10/14-15 人板——教练能识别并给出策略，但深度不如主流板子。

### 胜负判定

所有板子默认**屠边制**：狼杀光全部神或全部民即胜；好人公投出局全部狼即胜。

## 🎯 角色专属策略

教练会根据你的身份读取对应策略章节：

### 好人阵营

| 角色 | 核心策略 | 必杀技 |
|---|---|---|
| **预言家** ⭐ | 警徽流砸狼坑、饱满起跳、对跳应对 | 用警徽流策略逼狼面暴露 |
| **女巫** 💊 | 反向锁定（从狼刀读狼面）、藏身份、毒药决策 | "狼刀动作暴露狼队视角" |
| **猎人** 🔫 | 前期藏枪、翻牌时机、开枪条件 | "4 号在就藏，4 号死就翻" |
| **守卫** 🛡️ | 空守策略、双神盾配合、连守 | "守-救-守"模式保预言家 |
| **平民/白痴** 🧑 | 藏身份、抿狼、跟票、白痴翻牌后带队 | 表水让好人感受到你的好人视角 |

### 狼人阵营

| 角色 | 核心策略 | 必杀技 |
|---|---|---|
| **悍跳狼** | 模仿真预模板、报假查杀/金水、抢警徽 | 话术精到骗全场 |
| **冲锋狼** | 站边悍跳队友、攻击真预破绽 | 力度对比压制 |
| **倒钩狼** | 铁站真预、票队友做高身份、深水潜伏 | 决赛圈扛推好人 |
| **深水狼** | 全程平民伪装、不当焦点 | 活到最后屠边 |

### 警长（任何身份）

警上发言站位、警下归票权使用、警徽流管理、孤狼残局识别。

### 文件加载策略

教练**不会一次性读取所有文件**，而是按需读取（避免上下文爆炸）：

| 文件 | 何时读 |
|---|---|
| `SKILL.md` | 技能触发时必读 |
| `strategy.md` | **每局开始必读对应用户身份章节**；遇到对跳、盲毒、自爆等战术时读相关小节 |
| `rules.md` | 用户提到不熟悉的角色、板子，或出现规则争议时 |
| `glossary.md` | 用户用了不熟的术语，或你需要用术语给建议时 |

## ⚙️ 模式切换

### 默认：精简模式

适合**实战中使用**——快速扫完，每轮 30 秒可读完：

- 正文 ≤500 字
- 台词 ≤100 字（警上 ≤120）
- 玩家档案 ≤6 行
- 风险 ≤2 条
- 追问 ≤3 个

### 可选：详细模式

当你说**"详细点""展开""再细一点"**时切换，追加：

- 每个存活玩家都给身份概率+理由
- 票型分析展开（每个票阵营拆开）
- 资源使用建议（女巫救/毒、猎人开枪、守卫守谁、预言家验谁）
- 台词追加对照版（激进版/稳健版）

切换示例：
```
你：详细点分析这一轮
教练：[切换到详细模式，追加展开内容]
```

## ❓ FAQ

### Q1：线上语音局能用吗？还是只能线下面杀？

**都能用**。线上局你把听到的发言打字告诉教练即可；线下面杀你可以在每轮间隙偷偷看手机。

**注意**：实战中发言有时间限制（一般 1-2 分钟），所以台词严格控制在 ≤120 字，留出余地。

### Q2：教练能看到玩家的表情和语气吗？

**不能**。这是教练的局限。当判断高度依赖这些信息时，教练会主动问你：

> "X 平时喜欢悍跳吗？"
> "3 号刚才报查杀的时候，5 号的表情/反应是什么？"

你的回答越具体，教练的判断越准。

### Q3：我是狼人，教练会帮我吗？

**会**。教练根据你的身份服务——狼人用户会得到悍跳话术、刀人选择、自爆时机、狼队配合策略。但教练**不会在台词里泄露队友身份**（防泄露硬约束）。

### Q4：可以中途加入吗？比如第 2 天才开始用教练

**可以**。把之前发生的事简要告诉教练（"12 人预女猎守，我是女巫，第一天 3 号被公投出局，第二夜 4 号被刀..."），教练会基于当前状态开始分析。

### Q5：教练会保证我赢吗？

**不保证**。狼人杀是没有纯策略纳什均衡的博弈——任何套路都有克制之法。教练能做的是：
- 提高你的胜率（基于高手实战经验）
- 避免你犯典型错误（如盲毒、测试性投票、被话术精悍跳骗）
- 在关键节点给出最优决策

### Q6：板子不在支持列表里怎么办？

教练仍能工作，但策略深度可能不足。你可以告诉教练板子配置，它会基于规则库给出建议。如果遇到扩展角色（狼美人、骑士、熊等），教练会读取 `references/strategy.md §8` 扩展角色策略。

### Q7：教练的输出太长/太短怎么办？

- 太长 → 说"精简点"或"只给台词"
- 太短 → 说"详细点"或"展开分析"
- 只要台词 → 说"只给我该说什么"
